- 为了研究某一事件的规律，依据时间发生的顺序将事件在多个时刻的数值记录下来,就构成了一个时间序列，用$\{ Y _ { t } \} =$表示。
- Base R下的ts数据类型是专门为时间序列设计的，本质上是一个数值型向量,扩展了时刻属性使得每个数都有一个时刻与之对应。
- 用ts(data, start, end, freq, ...)函数创建一个时间序列。
- ts(data = 1: 10, start = 2010, end = 2019) #年度数据
- ts(data = 1: 10, start = 2010, frequency = 4) #季度数据

frequency 设置时间频率，默认为1,还可以为12，52，365

- fpp3生态下的tsibble包提供了整洁的时间序列数据结构tsibble.
- 时间序列数据，无非就是指标数据 + 时间索引(或者再 + 分组索引)
- 用as_tsibble()函数将数据框转换为tsibble数据结构。，只需指定时间索引(index)，分组索引(key)
